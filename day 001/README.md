## Day 001 - Python Strings Fundamentals

Today I learned:

* String concatenation using the `+` operator
* Creating full names from multiple variables
* Formatting strings with the `title()` method
* Using tabs (`\t`) to improve output formatting
* Using newlines (`\n`) to display text on multiple lines
* Understanding whitespace in strings
* Removing whitespace using:

  * `rstrip()`
  * `lstrip()`
  * `strip()`

### Example Concepts

```python
first_name = "ada"
last_name = "lovelace"

full_name = first_name + " " + last_name

print("Hello, " + full_name.title() + "!")
```

Key takeaway:

Small string methods can make programs look more professional and help clean user input before processing it.
